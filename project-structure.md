# Backend Directory (Node.js Express Boilerplate)

## Directory Structure:

- **/backend**
  - **/src**
    - **/controllers**      
    - **/models**           
    - **/routes**          
    - **/middleware**       
    - index.js             
- package.json
- server.js

## Essential Files:
- `package.json`: Contains project metadata and dependencies.
- `server.js`: Entry point for the application.
- Directory for controllers, models, routes, and middleware.
